# Бот-предложка для вашего телеграм канала

## Информация

Данный бот позволяет добавить функционал предложенных записей для вашего телеграм канала  
Поддерживает все типы каналов - от публичных до приватных

## Установка

```
git clone https://github.com/holy-jesus/Owlyfer # Чтобы склонировать этот репозиторий
```

Измените settings.ini, вставив туда bot_token и ID канала, после чего добавьте бота в канал, выдав ему право публикации сообщений

```
pip install -r requirements.txt # Чтобы установить все библиотеки
python3 utils/db_create.py # Чтобы создать базу данных
python3 main.py # Чтобы запустить бота
```
